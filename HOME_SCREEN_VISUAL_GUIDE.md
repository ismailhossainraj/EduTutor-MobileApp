## Key Interactions

### Bottom Navigation
- **Tap Tab** â†’ Switches between 4 main sections
- **Visual Feedback** â†’ Selected tab highlighted in blue
- **State Persisted** â†’ Remains on selected tab

### Action Buttons
- **On Tap** â†’ Navigate to specific screen
- **Visual Feedback** â†’ Color change on press
- **Icons** â†’ Indicate button purpose

### Logout
- **On Tap** â†’ Sign out from Firebase
- **Navigation** â†’ Return to LoginScreen
- **Confirmation** â†’ Optional implementation

### Profile Menu
- **On Tap** â†’ Navigate to respective screen
- **Arrow Indicator** â†’ Shows navigation available
- **Clear Labels** â†’ Descriptive menu items

## Responsive Behavior

- **Padding**: Consistent 16pt horizontal margins
- **Scrolling**: SingleChildScrollView for overflow content
- **Grid**: 2-column layout for stats (scales with device)
- **Spacing**: Proportional gaps using SizedBox
- **Text**: Uses Theme.of(context) for scalable typography
